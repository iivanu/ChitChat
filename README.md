## Quick Start

```bash
# Install dependencies
cd server
npm install

cd client 
npm install

# Start Express Server: http://localhost:3000
cd server
npm start

# Start dev Express Server
cd server
npm run dev

# Start Vue DevServer: http://localhost:8080
cd client
npm run serve

# Build for production (Will build into server/public, ready for deployment)
cd client
npm run build
```
